# Daily Session Template

Use this template to generate a new study session after the planner has selected the next day.

## Inputs

- Day number
- Theme
- Target grammar focus
- Target reading focus
- Target writing goal
- Target speaking goal
- Target vocabulary theme
- Known learner weaknesses to emphasize

## Session Structure

1. Start with one short grammar correction block.
2. Provide one reading task with one or two comprehension questions.
3. Introduce five to eight vocabulary items tied to the theme.
4. Ask for one writing task with a realistic VSTEP-style prompt.
5. Ask for one speaking task with follow-up questions.
6. Include listening only if it meaningfully fits the theme.
7. End with an evaluation and memory update.

## Style Rules

- Keep prompts practical and exam-relevant.
- Prefer short tasks that fit into 30 minutes.
- Reuse the learner's weak areas deliberately.
- Avoid repetition from previous sessions unless it is a planned review.

## Required Deliverables

- Session notes in `sessions/day-XX.md`
- Summary report in `reports/day-XX-report.md`
- Updates to long-term memory files when needed

## Planner Link

- The session planner should read `ROADMAP.md` and `progress.md` first.
- The planner should then fill this template with the target day, theme, and weaknesses.
